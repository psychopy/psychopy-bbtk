TPadButtonGroup
-------------------------------

To import TPadButtonGroup, you can either use::

    from psychopy_bbtk.tpad import TPadButtonGroup

or, any time after `psychopy.plugins.activatePlugins` has been called::
    from psychopy.hardware.bbtk import TPadButtonGroup
